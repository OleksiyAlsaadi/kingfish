April 18, 2014      by Weston

        Tomorrow, on April 19, We will be presenting project

Kingfish to AIME's Robotronica Convention by CSU Chico!

This will be an historic event for PV Mechatronics; this is our chance to

prove ourselves, and we'd love if you could make it. There, you can

speak to Lead Designer Ian McNanie, Lead Programmer Alex Elson,

and Lead Mechanic Weston Cramer. If you see us, be sure to say hello!

April 5, 2014      by Alex Elson

        We have a website! I've been working with HTML, along with Javascript, to

create a custom looking design. More pages should be added over time, and as we

take up new projects. We still need a members and sponsors page, though it shouldn't

be too long before then. At the same time, Ian McNanie is continuing to work on the

mechanical side of our latest project, which we chose to call "Kingfish". In a few months

time we hope to be able to travel to Colorado and perform in the 2014 Autonomous Vehicle

Competition, hosted by Sparkfun. I've already written the code to control object tracking

in OpenCV, so it shouldn't be too long before we'll ready to present it at the next Robotronica,

AIME's local robotics showcase at Chico State University.
